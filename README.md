This project implements a system for efficient delivery of government schemes using Blockchain technology. It uses a decentralized approach so records are maintained in blocks on a blockchain, improving transparency and immutability.

## What this repository contains

- Project source (smart contracts, backend, frontend) — see the repository folders for details.
- README with setup and contribution instructions.

## Key goals

- Provide a transparent ledger for government scheme delivery.
- Securely track disbursements and beneficiary records.
- Make audit and verification straightforward through immutable records.
